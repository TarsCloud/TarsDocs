# 性能数据

## 测试环境

* 测试机器类型：4 核 /8 线程 CPU 3.3Ghz，16G 内存
* 测试逻辑：客户端将一定数量的数据传输到服务器，然后服务器将其原样返回给客户端。
* 服务器单进程，多个客户端启动工作台。
